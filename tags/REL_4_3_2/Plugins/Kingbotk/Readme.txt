The Kingbotk plugin is a templating plugin for WikiProjects.

It is developed and supported by Kingboyk with help from Reedy Boy.

For more information please see http://en.wikipedia.org/wiki/User:Kingbotk/Plugin .

To use this plugin:
*Move or copy the DLL file to your AWB folder
*Read the instructions at the above page and watchlist it. We recommend adding your name to the user list too.
*Enable the templating plugins you wish to use, configure them, and proceed as normal.
